Base Usage Assumptions

The model uses explicit voice, scored-call, and software-floor inputs rather than hidden averages.

Live voice model
$0.08 / min

Internal planning conversion from OpenAI realtime token pricing, not an official vendor minute quote.

Scored call model
$0.325 / call

52.5 transcription minutes plus a small GPT-5 mini buffer. Solo models 20 calls monthly; Team minimum 100; Team 10-seat 150.

Shared software floor
$173.83 / mo

Vercel, Supabase, Fly, HighLevel, and Resend spread across 25 active paying orgs in the base case.

  • Solo is modeled at 20 scored calls per month.
  • Team minimum is modeled at 100 scored calls per month; the 10-seat reference case uses 150.
  • Optional gpt-4o-mini-tts playback is ~$0.015 per minute and excluded from base live voice metering.

Appendix: Formula Detail

The core formulas and planning conversions behind the seat model.

Key formulas
MetricFormulaOutput
Live voice planning costDerived internally from GPT-realtime-1.5 token pricing$0.08 / minute
Scored call planning cost52.5 min × $0.006 transcription + $0.01 GPT-5-mini scoring buffer$0.325 / call
Base scored-call loadSolo: 20 / month; Team minimum: 100 / month; Team 10-seat: 150 / monthPlan-specific org volume, not a flat per-seat average
Shared software floor$20 Vercel + $25 Supabase + $11.83 Fly + $97 HighLevel + $20 Resend$173.83 / month
Base-case floor allocation$173.83 / 25 active paying orgs$6.95 / org / month
Collected gross marginProduct gross margin minus Stripe processing and Stripe Billing feesStripe shown separately from product COGS
Internal underwriting ratesUsed only for pack sizing and margin guardrailsSolo 0.39 / min; Team 0.29 / min
Modeling notes
  • Live voice planning cost is derived from official OpenAI token pricing and presented as an internal minute conversion.
  • Product gross margin excludes Stripe. Collected gross margin subtracts Stripe processing and Stripe Billing fees.
  • Solo is modeled at 20 scored calls per month; Team minimum at 100; Team 10-seat at 150.
  • Internal underwriting rates remain $0.39/minute for Solo and $0.29/minute for Team, but public overage is sold through prepaid packs.
  • Purchased voice packs are modeled with 90-day validity in the current implementation plan.
